Installation
-------------------

For right now we suggest installing from GitHub because of the rapid development of this project

pip
^^^^^^^^^^^^^^^^^^^^^^
 
Releases are available PyPI and can be installed with pip

.. code:: console

    $ pip install chimeric-tools


You can alternatively install the latest version from GitHub

.. code:: console
   
      $ pip install pip install git+https://github.com/computationalUncertaintyLab/chimeric-tools.git
