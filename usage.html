Usage¶

Installation¶

To use chimeric_tools, first install it using pip:

$ pip install chimeric_tools

chimeric_tools.Spatial.GetStateAbbreviation(String)¶

Return the abbreviation of a state name

Param

StateName

Returns

StateAbbreviation

Return type

STRING

>>> from chimeric_tools import Spatial
>>> Spatial.GetStateAbbreviation("Pennsylvania")
"PA"
chimeric_tools.Spatial.fromState2FIPS(String)¶

Return the FIPS value that corresponds to the state (For example the FIPS for PA is 42)

Param

String abbrevation of a state. For example “PA”

Returns

The FIPS value that corresponds to the state (For example the FIPS for PA is 42)

Return type

INT

>>> from chimeric_tools import Spatial
>>> Spatial.fromState2FIPS("PA")
42

Param

None

Returns

the epiweek

Return type

INT

>>> from chimeric_tools import Time
>>> Time.todayEpiWeek()
12
chimeric_tools.Time.fromDate2Epiweek(String)¶

Return the epidemic week that corresponds to a date in YYYY-mm-dd format

Param

a string in the format YYYY-mm-dd

Returns

the epiweek

Return type

INT

>>> from chimeric_tools import Time
>>> Time.fromDate2Epiweek(fromDate2Epiweek("2020-03-22"))
13
chimeric_tools.Time.fromDates2Epiweeks(List)¶

Return the epidemic weeks that corresponds to a list of dates in YYYY-mm-dd format

Param

a list of strings in the format YYYY-mm-dd

Returns

a list of epiweeks

Return type

LIST

>>> from chimeric_tools import Time
>>> Time.fromDates2Epiweeks(["2020-01-22","2020-02-23","2020-03-24"])
[4, 9, 13]

This function crawls data from the delphi epidata API and stores it in a csv file.

Parameters

  • startepiweek – the first epiweek to crawl

  • endepiweek – the last epiweek to crawl

  • regions – the list of regions to crawl

return: a dataframe of the data

>>> Data.crawl_data_to_feature_specific(201201, 202201, ['nat'])
